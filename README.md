# ⏱️ Stopwatch Web Application

## SkillCraft Technology Internship - Task 02

An interactive and responsive Stopwatch Web Application developed using HTML, CSS, and JavaScript as part of the SkillCraft Technology Web Development Internship.

---

## 📌 Project Description

This project is a fully functional stopwatch application that allows users to accurately measure elapsed time. It includes essential stopwatch controls such as Start, Pause, Reset, and Lap recording.

The application features a modern and responsive user interface, ensuring a seamless experience across desktops, tablets, and mobile devices.

## 🎯 Task Objective

Build a stopwatch web application that:

- Accurately tracks elapsed time.
- Allows users to start, pause, and reset the stopwatch.
- Records lap times.
- Provides an intuitive and responsive user interface.
- Demonstrates JavaScript timing and DOM manipulation skills.

---

## ⚙️ How It Works

1. Click the Start button to begin timing.
2. Click Pause to temporarily stop the timer.
3. Click Start again to resume timing.
4. Click Lap to record the current elapsed time.
5. Click Reset to clear all recorded times and restart the stopwatch.
